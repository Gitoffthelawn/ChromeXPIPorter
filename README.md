> It will work with the regular version of Firefox, but additional steps are required.

# Tested browsers
* Floorp (Recommended)
* Firefox Nightly
* Firefox Developer Edition
* Librewolf
* Zen browser

***Except for Floorp, value of `xpinstall.signatures.required` have to set `false` in `about:config`.***
