How to use with Android FF

Complete guide for installing extensions on Android Firefox

In the Android version of Firefox nightly or Iceraven, clicking the 'Add to Firefox' button does not display an installation pop-up, but instead directly downloads the xpi file.

This document explains how to install it.

1. Download XPI file

When you try to install using CRX Installer, a patched XPI file will be downloaded.

2. Enable debug menu

Navigate to ⋮ → Settings → About Firefox Nightly (About Iceraven) and tap repeatedly on the Firefox logo.

3. Install extension

Tap Install extension from file added to the Advanced section of Settings and select the downloaded XPI file.
